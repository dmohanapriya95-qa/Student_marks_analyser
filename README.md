# 🎓 Student Marks Analyzer (Python Project)

## 📌 Description

This project takes marks of multiple subjects from the user and calculates:

* Total marks
* Average
* Percentage
* Maximum and Minimum marks
* Pass/Fail result

## 🚀 Features

* User input validation (0–100 marks only)
* Error handling for invalid input
* Loop-based logic using `for` and `while`
* Repeat option for multiple runs

## 🛠️ Technologies Used

* Python

## ▶️ How to Run

1. Open terminal
2. Run:

   ```
   python main.py
   ```

## 💡 Concepts Used

* for loop
* while loop
* if conditions
* exception handling
* user input
